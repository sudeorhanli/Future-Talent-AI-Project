# 🌱 GreenBudget AI: Sürdürülebilir Kamu Maliyesi Simülatörü

Bu proje, **Future Talent Program 201 - Yapay Zeka Bitirme Projesi** kapsamında geliştirilmiştir.

### 🎯 Projenin Amacı
Geleceğin ekonomi dünyasında **"Yeşil Vergiler"** ve **"Karbon Ayak İzi"** takibi kritik bir rol oynayacaktır. Kamu Maliyesi disiplini ile yapay zeka araçlarını birleştiren bu uygulama; bireysel harcamaların çevresel etkisini analiz eder ve potansiyel vergi yükümlülüklerini simüle eder.

### 🚀 Özellikler
- **Gerçek Zamanlı Karbon Analizi:** Ulaşım, enerji, su ve gıda harcamalarının emisyon hesaplaması.
- **Yeşil Vergi Simülasyonu:** Karbon salınımına dayalı dinamik vergi hesaplama motoru.
- **AI Destekli Öneriler:** Kullanıcı verisine göre kişiselleştirilmiş sürdürülebilirlik tavsiyeleri.

### 🛠️ Kullanılan Teknolojiler
- **Python** (Mantık katmanı)
- **Streamlit** (Arayüz ve Dağıtım)
- **Kural Tabanlı AI** (Karar Destek Mekanizması)
